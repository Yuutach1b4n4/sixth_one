Поиск круглых контуров с фильтрацией площади

Приложение для автоматического обнаружения круглых контуров на изображениях с возможностью динамической фильтрации объектов по минимальной площади и выгрузкой результатов.

## Возможности

- Загрузка исходного графического изображения любой структуры.
- Динамическая фильтрация контуров с помощью виджета Spinbox.
- Автоматический подсчет количества обнаруженных предметов.
- Интерактивный вывод результатов и счетчика объектов в заголовок окна.
- Сохранение результирующего обработанного изображения на жесткий диск.

## Установка зависимостей

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

## Запуск

python main.py
